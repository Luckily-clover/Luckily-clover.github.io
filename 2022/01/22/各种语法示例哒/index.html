tag-hide

如果想把一些文字、内容隐藏起来，并提供按钮让用户点击显示。可以使用这个标籤外挂。

line

inline 在文本里面添加按钮隐藏内容，只限文字

( content不能包含英文逗号，可用‚)

  1. content: 文本内容

  2. display: 按钮显示的文字(可选)

  3. bg: 按钮的背景颜色(可选)

  4. color: 按钮文字的颜色(可选)

哪个英文字母最酷？ 因为西装裤(C装酷)

block

block独立的block隐藏内容，可以隐藏很多内容，包括图片，代码块等等

( display 不能包含英文逗号，可用‚)

  1. content: 文本内容
  2. display: 按钮显示的文字(可选)
  3. bg: 按钮的背景颜色(可选)
  4. color: 按钮文字的颜色(可选)
